# nvim

Requires `neovim` >= `0.8.x`

## Binaries needed

`mason.nvim` will install LSPs, linteres & formatters. Some tools are required for it to do so.

### Required

- `wget`
- `node`
- `python3`
- `go`
- `rustup`
- `make`
- `gcc` or `clang`
- `fzf`
- `fd`
- `rg`

### Optional

- `bat`
- `delta`
